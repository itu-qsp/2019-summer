Pandas .loc()¶

Slices the dataset using both row and column indices:

df.loc[rows, columns]

For example:

  • df.loc[:, :]
  • df.loc[0, 1960:2018]

Great resource: https://medium.com/dunder-data/selecting-subsets-of-data-in-pandas-6fcd0170be9c
